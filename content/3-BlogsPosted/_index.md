Below are the blog posts I have researched and written, shared with the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) community.

### [Blog 1 - Building Memory-Intensive Apps with AWS Lambda Managed Instances](3.1-Blog1/)

This blog introduces AWS Lambda Managed Instances – a feature that allows running Lambda functions on Amazon EC2 instances with up to 32 GB RAM, enabling memory-intensive applications like AI/ML inference and in-memory analytics while maintaining the convenience of serverless architecture.

### [Blog 2 - Building Multi-Step Workflows with AWS Lambda Durable Functions](3.2-Blog2/)

This blog introduces AWS Lambda Durable Functions – enabling developers to write multi-step workflows within a single Lambda function, with automatic checkpointing, retry capabilities, and the ability to pause for up to one year without compute costs. Ideal for AI workflows, order processing, and human-approval processes.

### [Blog 3 - OpenSearch Serverless NextGen: Vector Search with Scale-to-Zero](3.3-Blog3/)

This blog introduces the next generation of Amazon OpenSearch Serverless (NextGen) – a new architecture that decouples compute and storage, enabling true scale-to-zero when idle and 20x faster scaling. An ideal solution for vector search backends powering AI agent applications.
